02. Analyse du Contexte

Green-Tech Solutions

PME innovante de 45 collaborateurs spécialisée dans l'énergie renouvelable. L'entreprise héberge en interne ses applications critiques (CRM, Site Vitrine, ERP).

Diagnostic : Réseau Plat

L'audit initial a révélé une faille architecturale majeure : le serveur Web public (accessible depuis Internet) est situé dans le même sous-réseau que les postes de la direction et des finances.

Scénario d'attaque possible :
Internet Serveur Web (Compromis) Serveur Compta (Ransomware)
Enjeu Sécuritaire

"Comment exposer les services web publics tout en garantissant l'intégrité absolue du réseau interne ?"

04. Cahier des Charges

1. Cloisonnement

Création de 3 zones étanches : WAN (Rouge), LAN (Verte), DMZ (Orange).

2. Filtrage Strict

Principe du "Deny All". Seuls les flux explicitement nécessaires sont autorisés.

3. Disponibilité

Le site Web doit rester accessible depuis l'extérieur via une IP unique (NAT).

08. Services d'Infrastructure

DHCP Server (LAN)

Distribution automatique pour le réseau interne uniquement. La DMZ reste en IP statique pour la sécurité.

Range: 192.168.10.100 - .200

DNS Resolver (Unbound)

Mode résolveur activé pour l'indépendance et la rapidité. Activation de DNSSEC pour l'intégrité.

09. Règles de Filtrage

C'est ici que se joue la sécurité. Nous définissons explicitement qui a le droit de parler à qui. La règle d'or : La DMZ ne doit JAMAIS initier de connexion vers le LAN.

10. Exposition NAT (Port Forwarding)

Le principe

Le serveur Web a une IP privée (172.16.50.10). Pour qu'il soit accessible depuis Internet, nous devons rediriger le trafic arrivant sur l'IP Publique (WAN) vers l'IP Privée.

Interface: WAN
Protocol: TCP
Destination Port: 80 (HTTP)
Redirect Target IP: 172.16.50.10 (SRV-WEB)
Redirect Target Port: 80 (HTTP)

11. Filtrage Applicatif (Squid)

Pourquoi un Proxy ?

Le pare-feu filtre les ports (Niveau 4). Le proxy Squid filtre les URLs et le contenu (Niveau 7). Cela permet de bloquer des sites spécifiques même s'ils utilisent le port 80/443.

Configuration

  • Mode Transparent (Pas de config client).
  • Blacklist : Jeux, Réseaux Sociaux, Warez.
  • Cache activé pour accélérer le web.
