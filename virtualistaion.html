Architecture Matérielle

Sizing & Pré-requis

Pour consolider 10 serveurs physiques, nous avons dimensionné un serveur hôte unique mais puissant, capable de supporter la charge cumulée avec une marge de 30% pour l'évolution future.

Serveur Hôte : DELL PowerEdge R740

  • CPU 2x Intel Xeon Gold 6248R (24 Cœurs)
  • RAM 128 Go DDR4 ECC Registered
  • Stockage 4x 2To SSD (RAID 5 Matériel)
  • Réseau 4x 10GbE SFP+ (Redondance)
03

Installation & Configuration

De la couche physique à l'OS

1. Préparation BIOS/UEFI

  • Activation de Intel VT-x / VT-d (Virtualization Technology).
  • Configuration du mode d'alimentation sur "Performance".
  • Désactivation des ports série/parallèle inutiles.

2. Configuration RAID (PERC)

  • Accès au contrôleur PERC H730P au démarrage (Ctrl+R).
  • Création d'un Virtual Disk (VD) en RAID 5 avec les 4 SSD.
  • Initialisation rapide du volume logique.

3. Installation ESXi

  • Boot sur clé USB bootable (Rufus + ISO ESXi 7.0).
  • Sélection du disque cible (Volume RAID Dell).
  • Définition du mot de passe Root (complexe).

4. Configuration Post-Install (DCUI)

  • Définition IP Statique de Management : 192.168.1.10/24.
  • Configuration DNS et Hostname (esxi-01).
  • Test du réseau de management.

Politique de Sécurité

Configuration par défaut du vSwitch pour la sécurité :

  • Promiscuous Mode : Reject (Empêche le sniffing).
  • MAC Address Changes : Reject (Empêche le spoofing).
  • Forged Transmits : Reject.

Hardening & Sécurité

Verrouillage de l'infrastructure

Lockdown Mode

Désactivation de l'accès direct console (DCUI) pour forcer l'administration via le client Web sécurisé.

Secure Boot

Utilisation de la puce TPM 2.0 du serveur Dell pour valider la signature cryptographique du noyau ESXi au démarrage.

Mots de passe

Politique de complexité renforcée pour le compte root (12 char, spéciaux, chiffres) et rotation mensuelle.

Stratégie de Sauvegarde

VEEAM
Backup & Replication

Mise en place de la règle du 3-2-1 pour garantir l'intégrité des données :
3 copies des données, sur 2 supports différents, dont 1 hors site.

  • Snapshots : Pris avant toute mise à jour critique (Windows Update, install soft).
  • Backup Quotidien : Incrémental à 22h00 sur le NAS.
  • Backup Hebdo : Full backup le Samedi vers le Cloud (S3).
